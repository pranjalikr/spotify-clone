# Spotify Clone 🎵

A Spotify-inspired frontend clone built using HTML and CSS while learning web development.

## 🚀 Features
- Sidebar navigation
- Music player UI
- Playlist cards
- Responsive layout

## 🛠 Tech Stack
- HTML5
- CSS3

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in browser
